## ğŸ§  **CramÃ©râ€™s V Correlation Explained in Simple Terms**

**CramÃ©râ€™s V** is a statistic used to measure the **association between two categorical variables**. It is based on the chi-square statistic and gives a value between:

* **0 (no association)**
* **1 (perfect association)**

Unlike chi-square which tells you **whether** two variables are related, **CramÃ©râ€™s V** tells you **how strongly** they are related.

---

### ğŸ“Œ **When to Use CramÃ©râ€™s V?**

* You have **two categorical variables** (like gender and product preference).
* You want to **measure the strength of association**, not just test for independence.

---

### ğŸ“ **Formula of CramÃ©râ€™s V**

$$
V = \sqrt{\frac{\chi^2}{n \cdot (k - 1)}}
$$

Where:

* $\chi^2$ = Chi-square statistic
* $n$ = total number of observations
* $k$ = min(number of rows, number of columns)

## Python program to compute **CramÃ©râ€™s V**
---

### ğŸ“Š **What the Plot Shows**

* Each **group** of bars represents a gender.
* Each **color** corresponds to a type of drink.

```python
import pandas as pd
import numpy as np
from scipy.stats import chi2_contingency

def cramers_v(confusion_matrix):
    chi2, p, dof, expected = chi2_contingency(confusion_matrix)
    n = confusion_matrix.sum().sum()
    min_dim = min(confusion_matrix.shape) - 1
    return np.sqrt(chi2 / (n * min_dim))

# Sample data as a contingency table
data = pd.DataFrame({
    'Coffee': [20, 10],
    'Tea': [30, 40],
    'Juice': [10, 20]
}, index=['Male', 'Female'])

# Compute CramÃ©râ€™s V
v = cramers_v(data)
print(f"CramÃ©r's V: {v:.4f}")
```
